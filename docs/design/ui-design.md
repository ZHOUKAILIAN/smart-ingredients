# UI Design - Smart Ingredients

## Design Philosophy

- **简洁优先** - 突出核心功能，减少干扰
- **视觉清晰** - 使用直观的颜色和图标传达信息
- **响应迅速** - 实时反馈用户操作
- **中文友好** - 针对中文用户优化布局

## Color Scheme

| 用途 | 颜色 | 说明 |
|--------|--------|------|
| 主色 | #10B981 | 绿色 - 健康安全 |
| 警告 | #F59E0B | 橙色 - 需要注意 |

## Page Overview

### 1. Home Page (首页)

- **标题**: "Smart Ingredients"
- **副标题**: "食品配料表分析助手"
- **主要操作**: 大按钮 "开始分析"
- **布局**: 居中，简洁

### 2. Upload Page (上传页)

- **相机按钮**: 大尺寸，易于点击
- **相册按钮**: 从设备选择图片
- **图片预览**: 上传前确认
- **文件限制**: 提示最大 10MB
- **支持格式**: JPEG, PNG, WebP

### 3. Processing Page (处理中)

- **进度指示器**: OCR 识别 → LLM 分析
- **状态文字**: "正在识别文字..." / "正在分析配料..."
- **预计时间**: 显示剩余等待时间

### 4. Result Page (结果页)

**顶部区域**:
- 健康评分: 大数字显示 (0-100)
- 评分颜色: 绿(>70) / 黄(50-70) / 红(<50)

**配料列表**:
- 按风险等级分组
- 每个配料: 名称 + 风险标签 + 说明
- 可展开查看详情

**警告区域**:
- 有害添加剂: 红色警告图标
- 过敏原: 橙色提醒
- 建议: 文字描述

**操作按钮**:
- 保存到历史
- 分享结果
- 重新分析

### 5. History Page (历史页)

- **列表展示**: 时间倒序
- **每项显示**: 缩略图 + 评分 + 时间
- **筛选**: 按评分/时间排序
- **搜索**: 按产品名称查找
- **收藏**: 标记常用产品

## Navigation

- **底部导航栏**: 首页 | 上传 | 历史
- **图标导航**: 使用直观图标而非文字
- **面包屑**: 显示当前页面路径

## Typography

- **中文字体**: PingFang SC, Microsoft YaHei
- **英文字体**: Inter, system-ui
- **标题**: 24px / 20px / 16px
- **正文**: 14px
- **行高**: 1.6 倍字号

## Interaction Patterns

### 加载状态
- 骨架屏: 内容区域显示占位
- 旋转动画: 中心加载图标
- 进度条: 多步骤任务显示

### 错误处理
- 友好提示: "图片上传失败，请重试"
- 错误图标: 红色感叹号
- 重试按钮: 一键重新操作

### 成功反馈
- 绿色对勾: 操作完成
- 自动跳转: 2秒后进入下一页
- 可取消: 用户手动跳过

## Accessibility

- **颜色对比**: WCAG AA 标准
- **键盘导航**: 所有操作可通过键盘完成
- **屏幕阅读**: 合适的 ARIA 标签
- **触摸目标**: 最小 44x44px

## Mobile Considerations

- **单手操作**: 主要按钮在拇指可达区域
- **横竖适配**: 自动调整布局
- **网络提示**: 离线状态显示
